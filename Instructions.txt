How to Access the Website Portfolio 

Ensure you have the following tools installed on your computer:

* A web browser (e.g., Chrome, Firefox, Safari).
* Xampp

Steps to Access 

1. Download the CSC541.zip file and extract the file.
2. Locate the Xampp file and paste the extracted file to the htdocs in the Xampp file in your folder. 
3. Make sure the Xampp server is online if not go to the Xampp and start the module for Apache and MySQL.
4. Once the server starts go to the web browser and type localhost/CSC541 and the portfolio website can be accessed.
