Portfolio Website

Purpose of the Website:
This is a personal portfolio website designed to showcase myself, my skills, my projects, and my contacts.

Technologies Used:
Frontend: HTML5, CSS3, JavaScript

Features and Functional:
About Me Section: Highlights my background, my goals, and strengths.
Project Gallery: Displays a collection of projects with descriptions, and technologies used.
Responsive Design: Optimized for all screen sizes, from mobile to desktop.
Contact Form: Enables visitors to send inquiries directly.
Social Media Integration: Links to professional profiles like LinkedIn, GitHub, Instagram, and WhatsApp.
Smooth Navigation: Scroll animations and an intuitive layout for seamless user experience
Responsive Web Design: Resize to the screen device like smartphones, tablets or laptop.
